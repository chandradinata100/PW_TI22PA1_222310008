Python

Pyton adalah Bahasa pemrograman serbaguna yang populer untuk pengembangan web, ilmu data, kecerdasan buatan, dan banyak lagi, dengan sintaksis yang mudah dipahami.

JavaScript

JavaScript adalah Bahasa pemrograman yang sering digunakan untuk mengembangkan aplikasi web interaktif dan dinamis di sisi klien.

Java

Java adalah Bahasa pemrograman yang digunakan secara luas untuk mengembangkan berbagai jenis aplikasi, dari perangkat lunak desktop hingga aplikasi Android.

C#

C# adalah Bahasa pemrograman yang dikembangkan oleh Microsoft, sering digunakan untuk pengembangan aplikasi Windows, permainan, dan aplikasi web.

C

C adalah Bahasa pemrograman yang kuat dan efisien, sering digunakan dalam pengembangan sistem operasi, perangkat lunak sistem, dan perangkat embedded.

C++

C++ adalah Bahasa pemrograman yang berorientasi objek dan merupakan ekstensi dari bahasa C, sering digunakan dalam pengembangan perangkat lunak tingkat rendah dan tingkat tinggi.

Go

Go adalah Bahasa pemrograman yang dikembangkan oleh Google, dirancang untuk kinerja yang tinggi dan skalabilitas, sering digunakan untuk pengembangan aplikasi server.

R

R adalah Bahasa pemrograman yang digunakan khususnya dalam analisis data dan statistik, dengan beragam paket dan fungsi untuk mendukung analisis data yang kompleks.

Swift

Swift adalah Bahasa pemrograman yang dikembangkan oleh Apple, digunakan untuk mengembangkan aplikasi iOS, macOS, watchOS, dan tvOS.

PHP

PHP adalah Bahasa pemrograman yang sering digunakan untuk pengembangan situs web dinamis, dengan kemampuan untuk berinteraksi dengan basis data dan menghasilkan konten web secara dinamis.
